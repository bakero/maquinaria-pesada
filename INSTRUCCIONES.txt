# INSTRUCCIONES DE USO
# MaquinarIA Pesada - flujo maestro largo
# =======================================

Archivo maestro del sistema:
    PODCAST_MASTER_SPEC.md

Scripts principales:
    generar_guion.py
    generar_episodio_v2.py
    producir_episodio.py

Entradas del generador de guion:
1. PDF del tema
2. PDF maestro completo con referencias
3. Contexto adicional opcional

Variables de entorno necesarias:
    OPENAI_API_KEY
    ELEVENLABS_API_KEY

Dependencias:
    pip install openai elevenlabs pydub httpx python-dotenv pdfplumber

Comandos principales:

1. Generar solo el guion:
    python generar_guion.py --ep EP001 --pdf "ruta\tema.pdf" --master-pdf "ruta\document_pdf (3).pdf"

2. Generar solo el audio desde un guion validado:
    python generar_episodio_v2.py --guion Guiones\EP001_guion_etiquetas_v1.txt --ep EP001

3. Ejecutar el pipeline completo:
    python producir_episodio.py --ep EP001 --pdf "ruta\tema.pdf" --master-pdf "ruta\document_pdf (3).pdf" --generar-musica

Reglas clave del flujo:
- El guion debe superar 1800 palabras reales sin cabeceras ni nombres de speaker.
- El guion debe incluir hook, intro de sonido, saludo con advertencia IA, 4-6 bloques, inserciones, cierre de conceptos y cierre final.
- El guion debe terminar con verificaciones generadas por el propio script.
- El episodio final debe durar aproximadamente 14-17 minutos.
- El guion sale en `Guiones`.
- El audio final y los logs salen en `episodios`.
