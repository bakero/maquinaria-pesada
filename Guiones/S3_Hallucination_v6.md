Un modelo de lenguaje puede inventar datos, citas y hechos históricos con total seguridad. Te los presenta como verdad absoluta, sin dudas ni advertencias.

Esto se llama alucinación. No es un error aleatorio, sino una consecuencia directa de cómo estos sistemas predicen tokens. Durante el entrenamiento, el modelo aprende a completar patrones estadísticos, no a verificar hechos reales.

Imagina que le preguntas por un estudio científico que no existe. El modelo genera nombres de autores, fechas, resultados. Todo suena creíble. Todo es falso. El sistema nunca consultó una base de datos ni accedió a internet. Solo predijo qué palabras encajan mejor después de tu pregunta.

Las alucinaciones son especialmente peligrosas porque el modelo no dice "no sé". Simplemente fabrica información con la confianza de quien sabe exactamente qué está diciendo.

Conocer este límite es crítico antes de usar cualquier IA generativa en contextos donde la precisión importa.
